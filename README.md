# bess

This is part of a vibe coding hackathon for accountants in the crypto industry.
My vision is to create a simple statement from the Subscan block explorer, hence BESS - Blockchain Explorer Simple Statement.

## Getting Started

Install dependencies and run the app:

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Deploy on Vercel

Push to GitHub and import this repository into Vercel for one-click deployment.
